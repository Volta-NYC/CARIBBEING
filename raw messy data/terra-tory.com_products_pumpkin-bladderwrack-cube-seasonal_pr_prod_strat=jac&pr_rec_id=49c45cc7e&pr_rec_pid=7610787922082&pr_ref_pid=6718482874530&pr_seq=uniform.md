Smells heavenly and has done wonders for my skin

I received a free sample of this soap with my regular soap purchase (Gentle Baby Oats Cube). The first thing I noticed was the smell. I'm not normally into pumpkin spice, but the smell of this soap is lovely. It stays on the skin for a decent amount of time, but isn't overpowering. Not only does it smell amazing, the changes I've seen in my skin since I've started using it have also been amazing. I've been using it as a body soap. I have very dry and bumpy skin thanks to KP. I've been using the smaple for 4 days now instead of my typical drug store brand soap and my skin has become so soft and smooth. I had to splurge and buy a full cube. My only small complaint is that the dried bladderwrack can be a bit prickly and scratchy. That being said, I'm obsessed with this soap and I hope it comes back as a regular seasonal item for this time of year.
